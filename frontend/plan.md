# 📌 Task Management System - Development Plan

## 🎯 Objective

Build a full-stack Task Management Application where users can register, log in, and manage their tasks efficiently.

---

## 🧱 Project Structure

### 🔹 Frontend (React)

* React (UI)
* Redux Toolkit (State Management)
* React Router (Routing)
* Axios (API calls)

### 🔹 Backend (Node.js + Express)

* Express.js (Server)
* MongoDB + Mongoose (Database)
* JWT (Authentication)
* bcrypt (Password hashing)

---

## ⚙️ Features to Implement

### 🔐 Authentication

* User Registration
* User Login
* JWT Token generation
* Store token in localStorage
* Protected routes (Dashboard access only for logged users)

---

### 📋 Task Management

* Create Task
* View Tasks
* Update Task
* Delete Task
* Toggle Task (Done / Undone)

---

### 🎨 UI / UX

* Centered dashboard layout
* Clean and responsive design
* Loading and error handling

---

## 🔄 Workflow

1. Setup backend server and database
2. Create User and Task models
3. Implement authentication APIs
4. Implement task CRUD APIs
5. Setup React frontend
6. Create login and register pages
7. Connect frontend with backend using Axios
8. Setup Redux for state management
9. Build dashboard UI
10. Implement task features (CRUD)
11. Add route protection
12. Test full application

---

## 🚀 Running the Project

### 🔹 Backend

```bash
cd backend
npm install
npm start
```

### 🔹 Frontend

```bash
cd frontend
npm install
npm start
```
